## Getting Started
### Inside frontend folder: 
1. `npm install`
2. `npm start`

### Inside App folder: 
1. create a virtual enviroment
2. install requirements.txt in the venv using: `pip install -r requirements.txt`
3. run django server via `python manage.py runserver`
